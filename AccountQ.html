Account Questions

Can our company have multiple users on a single account?

Yes. On the Web App, find “Applications” on the dropdown menu under “My Account”, and follow the steps there to allow others to upload files to your account.


Can I use the same email for multiple API keys?

You need a unique email and password for one API account. If you need multiple API keys, you will need to link separate emails for each.


When does my API key expire?

Your key is a long-lived credential that will expire in 3 years. (In the future we will have additional options around key generation).


How should our team set up the email for our shared account?

Since the email will have the ability to change and recover the password to the account, the most typical setup would be to have a system email used where anyone who needs password access has access to that email.
