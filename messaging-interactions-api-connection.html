LivePerson Connection "How-to"

    Pre-requisite: Create a LivePerson Conversational Cloud Account

  • Sign up for a Conversational Cloud account
  • Get your Base URI with the Domain Retrieval Tool
  • Go to Campaign Builder/Data Sources/API/MI API/Edit to get API keys and secrets

    Connect to the LivePerson MI API in your VoiceBase Account

  • In your VoiceBase account go to Manage Data/Connections/Add Connection
  • Choose LivePerson Messaging from the Vendor dropdown menu
  • Create a connection name and fill in Name field
  • Fill in Account ID, Account URL, and keys and secrets from your LivePerson account
  • Note that the time may not be set retroactively. Please set the date to the current date.
