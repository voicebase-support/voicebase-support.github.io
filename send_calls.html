"How-to" Send audio to VoiceBase

S3 Ingress

  • Use VoiceBase's S3 Ingress option to upload files to our platform directly from your AWS S3 bucket. Step by step instructions may be found here.

Upload through the VoiceBase API

  • In your VoiceBase account go to Settings > Bearer Tokens > Add Bearer Token to create a Bearer Token for your account.
  • For API call examples in cURL, see the VoiceBase documentation.
  • "How-to" configuration examples for all features available on the VoiceBase platform are also included with the documentation.
  • Sample bulk upload Python scripts for upload from your local machine, URLS, or s3 bucket may be accessed here.
