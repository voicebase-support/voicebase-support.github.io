VoiceBase API

How do I retrieve processed data?

VoiceBase offers both polling and callback options to check for processing completion. Data can be returned in the callback or retrieved in a separate API request. Responses are in JSON format.


How will API usage be billed?

Our standard contract has a minimum of half a minute per recording. We bill for summarized usage. Please contact the sales team if you would like additional information on your particular use case.

Where can I find a record of my uploaded files?

Making a GET request on /media will retrieve a list of the media files you have uploaded to VoiceBase. You can find sample code for this process in the developer console.
