VoiceBase API

Which format is required for dual-channel recordings?

Any format is accepted for stereo recordings where parties are separated by channel. We currently support stereo. We do not support multi track recordings.


What file formats should be sent for video transcription?

We recommend that customers strip out the audio from video files and only send the audio to VoiceBase. We then send JSON data and time-stamped keywords back and the synchronization is done on the customerâ€™s end. (When a customer does upload video, we do transcoding.)


Does the API allow custom data to be passed in such as my internal IDs?

Yes, use the metadata attachment - any customer supplied metadata may be uploaded along with a media file.


Can I embed your player plug-in?

We can absolutely take our player plugin UI and embed it in your platform so that it appears native. Contact a Sales Engineer for more information.
